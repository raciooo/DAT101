  1. In this assignment, we use the DOM. Explain how this library can help us develop interactive web pages.

    2. Technically this is not a library but an API. Regardless, we can use the DOM to allow JavaScript to grab, manipulate, add or delete elements, classes, and behaviors within an HTML file. This adds dynamic behavior and interactivity

  2. What is an event? Simply explain what happens when we add an event listener to an object.

    3. An event is simply a thing that happens. The way we utilize events is by adding eventlistener, JavaScript will then run code based off of different events that take place, for example something being loaded, the clock of a key button or a mouse, or the position of amouse

  3. Sometimes we need to use an event object to get data from the event. Explain how this event object is structured, don't explain all functions or attributes. Try to explain, for example, how to retrieve data from a keyboard or mouse.

    4. Event objects are created when certain events occures, adding eventlisteners then passes the event into a function that runs upon retrieving this information. An example of this can be the location of a mouse, or which key has been pressed, affecting how the event function runs.

  4. Explain the main difference between DOM id and DOM name. Can we have identical IDs?

    5. IDs are used to identify a unique element in the HTML file. There can not be more than one object with the same ID. A name meanwhile is used to refer to all elements with that name, making it easier to group elements together.
